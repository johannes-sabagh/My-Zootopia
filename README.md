# Animal Web Generator

Enter an animal name, get a nice HTML page with facts about it. That's pretty much it.

## Installation & How to use

1. Get a free API key from api-ninjas.com
2. Create a ".env" file and add your key:
   
   API_KEY=your_key_here
   
3. Install the dependencies:
   
   pip install -r requirements.txt
   
4. Run it:
   
   -python animals_web_generator.py (Windows)
   -python3 animals_web_generator.py (Mac an Linux)
   
5. Enter an animal to look for

6. Open the generated "animals.html" in your browser

## Notes

- If the animal doesn't exist, the page will tell you so
- Only the first location is shown per animal
